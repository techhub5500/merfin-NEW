# ⚙️ Core do Sistema

## Propósito
Componentes fundamentais que suportam o funcionamento do sistema de agentes. Infraestrutura crítica.

## Estrutura
- **toolContext/** - Sistema de cache de sessão (ToolContext)
- **streaming/** - Mecanismo de streaming de eventos para UX

## Responsabilidades
1. Gerenciar estado da sessão entre agentes
2. Implementar cache inteligente com TTL
3. Fornecer streaming de eventos em tempo real
4. Garantir rastreabilidade completa de execuções
